FACTORIAL

Syntax: y = FACTORIAL(x)

The FACTORIAL function is an element by element function that returns the factorial of the argument. The argument can be a scalar, vector or matrix. A scalar argument results in a scalar. A vector argument results in a vector with the same length as the argument, and a matrix (tensor) argument results in a matrix (tensor) with the same dimensions as the argument.

The integer portion of the argument value is used to calculate the factorial. For example, if n = INT(x), then FACTORIAL(x) = n(n-1)(n-2)...(2)(1).
