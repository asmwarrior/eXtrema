CONVOLUTE function

Syntax: yc = CONVOLUTE(y,b)

The CONVOLUTE function accepts two vector arguments. It convolutes the first argument vector, y, with the specified blurring vector, b. The result is a vector the same length as y. A different process is used depending on whether the length of b is even or odd.

The DECONVOLUTE function calculates the deconvolution (but only for an even number of points).
