RAN

Syntax: vout = RAN( v )

The RAN function is an element by element function that requires one argument. The argument can be a scalar, vector or matrix. A scalar argument results in a scalar. A vector argument results in a vector with the same length as the argument, and a matrix (tensor) argument results in a matrix (tensor) with the same dimensions as the argument. RAN uses the current value of the seed to generate the next random number, 0 < RAN(v) < 1. The initial value for the random number seed is 12345. Every time a random number is requested, either from the GENERATE/RANDOM command or from the RAN function, the seed is updated.
