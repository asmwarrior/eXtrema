Kurtosis

Syntax: s = KURT(x)
s = KURT(w,x)

The KURT function only accepts one or two vector arguments. If only one vector is entered, the function returns the kurtosis of that vector's data. If two vectors are entered, the first vector holds the weights and the second vector holds the data. The result is always a scalar value.

The kurtosis, kurt, is a nondimensional quantity which measures the relative peakedness or flatness of a distribution, relative to a normal distribution.
