Contour labels

By default, contours are labeled with the actual contour level, using three significant digits. See the discussion on the \LEGEND qualifier for an alternate contour labeling facility.

Contour label size

The size of the contour labels is CONTOURLABELHEIGHT. If labels are not desired on the contours, set the CONTOURLABELHEIGHT characteristic to 0 to set the label height to zero before entering the CONTOUR command. The default is %CONTOURLABELHEIGHT = 2.

Contour label separation

The separation between contour labels is controlled with CONTOURLABELSEPARATION. The default is CONTOURLABELSEPARATION = 10.
