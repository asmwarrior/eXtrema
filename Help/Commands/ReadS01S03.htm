Example 3

The macro below reads data into vectors allowing
DUM.DAT is below the user to choose when to stop reading
--------------- ---------------------------------------------
10 20 30
40 50 60
70 80 90
this is a test
100 200 300
400 500 600
700 800 900
test line two
-10 -20 -30
-40 -50 -60
-70 -80 -90		 LNUM = 1
START:
READ DUM.DAT\LNUM X Y Z
IF ~EXIST('X') THEN GOTO DONE ! end of file
LIST X Y Z
ANS='Y'
INQUIRE 'read again ? (Y|n)' ANS
IF NES(UCASE(ANS),'Y') THEN GOTO DONE
LNUM = LEN(X)+LNUM+1
GOTO START
DONE:
