Maximum and number of points

Syntax: GENERATE x min ,, max npts

If the maximum value and the number of points are given, the increment is calculated: inc = (max-min)/(npts-1) and then the formula:

x[i] = min+(i-1)*inc    for    1 ≤ i ≤ npts

is applied.

For example, after the command:

GENERATE X -1 ,, 2 4

the vector X will be [-1; 0; 1; 2].
