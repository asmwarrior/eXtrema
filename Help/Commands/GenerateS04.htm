Random values

Syntax: GENERATE\RANDOM x min max npts

If the \RANDOM qualifier is used, x will be filled with npts random numbers that fall between min and max.

No increment should be given.

For example, after the command:

GENERATE\RANDOM X 1 2 5

the vector X will be [1.198525; 1.897874; 1.238289; 1.367985; 1.381705]
