HARDCOPY command

Syntax: HARDCOPY filename
Qualifiers: \POSTSCRIPT, \JPEG, \PNG, \WMF, \EMF
Defaults: \POSTSCRIPT
Examples: HARDCOPY outfile.eps
HARDCOPY\PNG outfile.png
HARDCOPY\PNG outfile.png 450 220

The HARDCOPY command is used for saving the graphics to a file. The available graphics formats are: PostScript, PNG, Windows metafiles, and Enhanced Windows metafiles. These formats are chosen by qualifier.

Qualifiergraphic format
\POSTSCRIPT  Encapsulated PostScript (default)
\PNGPortable Network Graphics
\WMFWindows Metafile
\EMFWindows Enhanced Metafile

For PNG output only, the width and height of the output file can be specified. For example, HARDCOPY\PNG filename 450 220, produces a PNG file which is 450 pixels wide and 220 pixels high. If the width and height are not entered, the PNG file will have the same size as the graphics window.
