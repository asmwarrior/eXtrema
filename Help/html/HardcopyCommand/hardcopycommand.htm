HARDCOPY command

Syntax: HARDCOPY filename
Qualifiers: \POSTSCRIPT, \JPEG, \PNG
Defaults: \POSTSCRIPT
Examples: HARDCOPY
HARDCOPY\PNG

The HARDCOPY command is used for saving the graphics to a file. The available graphics formats are: PostScript, JPEG, and PNG. These formats are chosen by qualifier.

Qualifiergraphic format
\POSTSCRIPT  Encapsulated PostScript (default)
\JPEGJoint Photographic Experts Group
\PNGPortable Network Graphics
