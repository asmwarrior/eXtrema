PLOTSYMBOLANGLE

Syntax: GET PLOTSYMBOLANGLE {a}
Default value: PLOTSYMBOLANGLE = 0

PLOTSYMBOLANGLE controls the angle of the plotting symbols drawn when the GRAPH command is entered. PLOTSYMBOLANGLE is the angle, in degrees, measured counterclockwise, between the x-axis and a plotting symbol.
