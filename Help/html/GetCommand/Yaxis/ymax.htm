YMAX

Syntax: GET YMAX {n}
Default value: YMAX = 10.0

YMAX controls the maximum value to be displayed on the y-axis.

If |YLOGBASE| > 1.0, then YMAX is assumed to be the exponent for the maximum value on the y-axis. The maximum value displayed on the y-axis is |YLOGBASE|YMAX . Integer exponents are always used for the axis numbering, so the y-axis may not end on a large tic mark. If |YLOGBASE| ≤ 1.0, YMAX is the actual maximum value for the y-axis, and the y-axis always ends on a large tic mark.
