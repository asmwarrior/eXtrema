YLEADINGZEROS

Syntax: GET YLEADINGZEROS {n}
Default value: YLEADINGZEROS = 0

YLEADINGZEROS controls whether leading zeros are displayed on the y-axis numbers.

YLEADINGZEROS = 0 → numbers displayed on the y-axis will not have leading zeros if they are between 0 and 1
YLEADINGZEROS ≠ 0 → numbers displayed on the y-axis will have leading zeros if they are between 0 and 1

The default is to not display these leading zeros. Numbers that are between -1 and 0 always have the leading zero displayed.
