YVIRTUALMIN

Syntax: GET YVIRTUALMIN {n}
Default value: YVIRTUALMIN = 0.0

YVIRTUALMIN controls the virtual minimum value for the y-axis. If YVIRTUALMIN is less than YMIN, the actual y-axis minimum value will be YMIN, while the large, numbered, tic marks will begin at YVIRTUALMIN, which may be 'off the end' of the y-axis. YVIRTUALMIN is always set to YMIN whenever YMIN is set, so you must set YVIRTUALMIN after YMIN.
