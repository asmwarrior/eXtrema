HISTORYSHOW

Syntax: GET HISTORYSHOW {s}

HISTORYSHOW controls how many lines of history to display for each numeric variable as a result of the SHOW command.

HISTORYSHOW < 0 → all stored history lines will be displayed
HISTORYSHOW = 0,1 → only the latest history line will be displayed
HISTORYSHOW > 0 → a maximum of HISTORYSHOW lines of history will be displayed for each variable
