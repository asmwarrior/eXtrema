AREAFILLCOLOR

Syntax: GET AREAFILLCOLOR { c }

AREAFILLCOLOR can be set to a scalar value or to a fixed color name, or to a numeric vector. If it is set to the value zero, the default, then area fill color is turned off. If it is set to a non-zero scalar value or a fixed color name, or to a numeric vector, then subsequent graphs, using the GRAPH command, will have the area under the curve, or under the histogram, filled with solid color(s).

HISTOGRAMTYPE = 0 (default)  →  the GRAPH command produces normal line graphs
the area under a curve can be filled with a solid color
HISTOGRAMTYPE = 1 (or 3)  →  the GRAPH command produces a histogram profile without tails
the area under this profile can be filled with a solid color
HISTOGRAMTYPE = 2 (or 4)  →  the GRAPH command produces a histogram profile with tails to x=0 (or y=0)
the area of each individual histogram bar can be filled with a specific solid color
