ASPECTRATIO

Syntax: GET ASPECTRATIO { r }

Use GET ASPECTRATIO r to put the value of the aspect ratio of the graphics page into scalar variable r. If the optional variable r is not entered, the aspect ratio value will be displayed in the command history and message window.

The aspect ratio of the graphics page is the height of the page divided by the width of the page. For example, SET ASPECTRATIO 8.5/11 sets the aspect ratio to be typical letter size in landscape orientation, while SET ASPECTRATIO 11/8.5 sets the aspect ratio to be letter size in portrait orientation.
