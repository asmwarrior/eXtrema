XNUMBEROFDECIMALS

Syntax: GET XNUMBEROFDECIMALS {n}
Default value: XNUMBEROFDECIMALS = 1

XNUMBEROFDECIMALS controls the number of digits to be displayed in the fractional parts of the numbers labeling the x-axis.

XNUMBEROFDECIMALS
-1 → display the numbers labeling the x-axis as integers, with no decimal point
 0 → display the numbers labeling the x-axis with no fractional part, but with a decimal point
> 0 → display the numbers labeling the x-axis with XNUMBEROFDECIMALS digits in the fractional part
