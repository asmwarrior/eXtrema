XTICSBOTHSIDES

Syntax: GET XTICSBOTHSIDES {n}
Default value: XTICSBOTHSIDES = 0

XTICSBOTHSIDES controls whether or not tic marks, both large and small, are drawn on both sides of the x-axis.

XTICSBOTHSIDES = 0 → only draw tic marks on one side of the axis line. XTICANGLE controls the side on which the tic marks will appear
XTICSBOTHSIDES ≠ 0 → draw tic marks on both sides of the axis line
