XPOWER

Syntax: GET XPOWER {a}
Default value: XPOWER = 0.0

XPOWER, along with XPOWERAUTO , controls the x-axis scale value that is appended to the x-axis text label. This scale value is a power of ten, that is, 10XPOWER, and the numbers labeling the x-axis should be multiplied by this scale value to get the correct graph units.
