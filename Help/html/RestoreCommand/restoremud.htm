MUD format data files

Syntax: RESTORE\MUD file
RESTORE\MUD\TITLE file
Qualifiers: \MUD, \TITLE

The RESTORE\MUD command restores muSR MUD type data files containing histograms. Histogram number j is stored in column j of matrix IH. Histogram number j can be extracted into a vector using: HJ = IH[*,j].

If the \TITLE qualifier is used, only the title is restored, into a character variable named RUNTITLE (so as not to confuse it with variable MUDTITLE used to store the same title on a complete restore).
