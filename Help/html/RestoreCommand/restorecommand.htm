RESTORE command

Syntax: RESTORE file
RESTORE\MUD file
RESTORE\MUD\TITLE file
Qualifiers: \MESSAGES, \MUD, \TITLE
Defaults: \MESSAGES

The RESTORE command restores a previously saved program session, saved with the SAVE command. This means that all variables will be restored, as well as the command history and message window information. Any displayed graphics will also be restored.
