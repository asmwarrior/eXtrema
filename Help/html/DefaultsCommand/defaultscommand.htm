DEFAULTS command

Syntax: DEFAULTS

The DEFAULTS command resets most of the graphics defaults.

Autoscaling the graph axes is turned on. The graph box is turned off. The curve type is set to normal curve (non-histogram) and the data curve line type and the line thickness are both set to 1. The plotting symbols are turned off, the plotting symbol size is set to 2%, the plotting symbol angle is set to 0, and the plotting symbol color is set to black.
