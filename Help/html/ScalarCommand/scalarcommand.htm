SCALAR command

Syntax: SCALAR s1 { s2 ... }
Qualifiers: \FIT
Examples: SCALAR A
SCALAR\FIT A B

This command defines s1 to be a scalar. Other variable names, s2 ..., may be included, and they will also be defined to be scalars. If sI already is a scalar, its value will be unchanged. If sI is an existing variable, but not a scalar, it will be destroyed first. If sI does not exist, it will be created and initialized to 1.
