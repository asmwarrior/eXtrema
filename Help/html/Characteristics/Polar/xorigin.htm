POLARORIGINX

Syntax: SET POLARORIGINX n
SET %POLARORIGINX n
GET POLARORIGINX { n }
GET %POLARORIGINX { n }
Default: %POLARORIGINX = 50

POLARORIGINX controls the horizontal coordinate of the polar plot origin. There are two forms:

 POLARORIGINX → the horizontal coordinate, in world coordinate units, of the origin of the polar plot
%POLARORIGINX → the horizontal coordinate, expressed as a percentage of the width of the graphics sub-window, that is, POLARORIGINX = %POLARORIGINX * (XUPPERWINDOW- XLOWERWINDOW)/100.
