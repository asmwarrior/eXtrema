POLARNAXES

Syntax: SET POLARNAXES n
GET POLARNAXES { n }
Default: POLARNAXES = 4

POLARNAXES controls the number of axes drawn on a polar plot. The default value is 4. The axis at 0° is drawn if POLARNAXES is 1. If POLARNAXES is greater than 1, the axis at 0° is drawn and POLARNAXES-1 other axes are drawn equally spaced about 360°, but without labels. For example, if POLARNAXES is 3, the axis at 0° is drawn and 2 other axes are drawn at angles of 120° and 240°.
