POLARNUMBERSANGLE

Syntax: SET POLARNUMBERSANGLE n
GET POLARNUMBERSANGLE { n }
Default: POLARNUMBERSANGLE = 0

POLARNUMBERSANGLE controls the angle of the base line for the numbers labeling the polar axis. POLARNUMBERSANGLE is the angle, in degrees, measured counterclockwise, between a line parallel to the polar axis and the base line of a number.
