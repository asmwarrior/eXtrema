POLARAXISCOLOR

Syntax: SET POLARAXISCOLOR n
SET POLARAXISCOLOR colorname
GET POLARAXISCOLOR { n }

Note: The GET POLARAXISCOLOR command always returns the color index, and not the color name, even if POLARAXISCOLOR was set with a color name.

POLARAXISCOLOR controls the color of the polar axis. The parameter c can be a numeric scalar or one of the fixed color names.

For example:

SET POLARAXISCOLOR 2 uses the colormap index 2
SET POLARAXISCOLOR RED uses the fixed color name "RED"
