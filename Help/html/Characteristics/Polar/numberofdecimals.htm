POLARNUMBEROFDECIMALS

Syntax: SET POLARNUMBEROFDECIMALS n
GET POLARNUMBEROFDECIMALS { n }
Default: POLARNUMBEROFDECIMALS = 1

POLARNUMBEROFDECIMALS controls the number of digits to be displayed in the fractional parts of the numbers labeling the polar axis.

If POLARNUMBEROFDECIMALS = -1 display the numbers labeling the polar axis as integers, with no decimal point

If POLARNUMBEROFDECIMALS = 0 display the numbers labeling the polar axis with no fractional part, but with a decimal point.

If POLARNUMBEROFDECIMALS > 0 display the numbers labeling the polar axis with POLARNUMBEROFDECIMALS digits in the fractional part.
