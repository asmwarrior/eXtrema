POLARAXISLENGTH

Syntax: SET POLARAXISLENGTH n
SET %POLARAXISLENGTH n
GET POLARAXISLENGTH { n }
GET %POLARAXISLENGTH { n }
Default: %POLARAXISLENGTH = 30

POLARAXISLENGTH controls the length of the polar axis. There are two forms:

 POLARAXISLENGTH → the length, in world coordinate units, of the polar axis
%POLARLARGETICLENGTH → the length, expressed as a percentage of the height of the graphics sub-window, that is, POLARAXISLENGTH = %POLARAXISLENGTH * (YUPPERWINDOW- YLOWERWINDOW)/100.
