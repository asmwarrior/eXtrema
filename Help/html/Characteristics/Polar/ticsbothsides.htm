POLARTICSBOTHSIDES

Syntax: SET POLARTICSBOTHSIDES n
GET POLARTICSBOTHSIDES { n }
Default: POLARTICSBOTHSIDES = 0

POLARTICSBOTHSIDES controls whether or not tic marks, both large and small, are drawn on both sides of the polar axis.

POLARTICSBOTHSIDES = 0 → only draw tic marks on one side of the axis line. POLARTICANGLE controls the side on which the tic marks will appear
POLARTICSBOTHSIDES ≠ 0 → draw tic marks on both sides of the axis line
