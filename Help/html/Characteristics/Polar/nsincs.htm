POLARNSINCS

Syntax: SET POLARNSINCS n
GET POLARNSINCS { n }
Default: POLARNSINCS = 0

POLARNSINCS controls the number of small, unlabeled, tic marks to be displayed between the large, labeled, tic marks on the polar axis.

If POLARNSINCS ≤ 1 draw no small tic marks on the polar axis.

If POLARNSINCS ≥ 2 the number of small tic marks on the polar axis, between each pair of large, labeled, tic marks, will be POLARNSINC-1.
