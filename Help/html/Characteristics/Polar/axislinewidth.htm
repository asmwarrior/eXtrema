POLARAXISLINEWIDTH

Syntax: SET POLARAXISLINEWIDTH n
GET POLARAXISLINEWIDTH { n }
Default: POLARAXISLINEWIDTH = 1

POLARAXISLINEWIDTH controls the line width of all the polar radial axes.

The units of POLARAXISLINEWIDTH are 1/4 pts, where a pt is 1/72 of an inch. For example, a line width of 1 gives 1/288 inch wide lines, while a line width of 4 gives 1/72 inch wide lines.

The parameter n must be a scalar.
