POLARDROPLASTNUMBER

Syntax: SET POLARDROPLASTNUMBER n
GET POLARDROPLASTNUMBER { n }
Default value: POLARDROPLASTNUMBER = 0

POLARDROPLASTNUMBER is another control on the numbering for the polar axis.

If POLARDROPLASTNUMBER = 0, the last, highest value, numeric label will be displayed on the polar axis

If POLARDROPLASTNUMBER ≠ 0, the highest value numeric label will not be displayed.
