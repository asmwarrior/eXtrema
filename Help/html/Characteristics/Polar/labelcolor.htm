POLARLABELCOLOR

Syntax: SET POLARLABELCOLOR n
SET POLARLABELCOLOR colorname
GET POLARLABELCOLOR { n }
Default: POLARLABELCOLOR = black

Note: The GET POLARLABELCOLOR command always returns the color index, and not the color name, even if POLARLABELCOLOR was set with a color name.

POLARLABELCOLOR controls the color of the automatic polar axis text label. The parameter c can be a numeric scalar or one of the fixed color names. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names.

For example:

SET POLARLABELCOLOR 2 uses the colormap index 2
SET POLARLABELCOLOR -2 uses the fixed color index 2
SET POLARLABELCOLOR RED uses the fixed color name "RED"
