POLARTICANGLE

Syntax: SET POLARTICANGLE n
GET POLARTICANGLE { n }
Default: POLARTICANGLE = 90

POLARTICANGLE controls the angle of the tic marks, both large and small, on the polar axis. POLARTICANGLE is the angle, in degrees, measured counterclockwise, between the polar axis and a tic mark.
