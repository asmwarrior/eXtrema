POLARNUMBERSCOLOR

Syntax: SET POLARNUMBERSCOLOR n
SET POLARNUMBERSCOLOR colorname
GET POLARNUMBERSCOLOR { n }
Default: POLARNUMBERSCOLOR = black

Note: The GET POLARNUMBERSCOLOR command always returns the color index, and not the color name, even if POLARNUMBERSCOLOR was set with a color name.

POLARNUMBERSCOLOR controls the color of the numbers labeling the polar axis. The parameter c can be a numeric scalar or one of the fixed color names. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names.

For example:

SET POLARNUMBERSCOLOR 2 uses the colormap index 2
SET POLARNUMBERSCOLOR -2 uses the fixed color index 2
SET POLARNUMBERSCOLOR RED uses the fixed color name "RED"
