POLARTICSON

Syntax: SET POLARTICSON n
GET POLARTICSON { n }
Default: POLARTICSON = 1

POLARTICSON controls whether or not tic marks, both large and small, are drawn on the polar axis.

POLARTICSON = 0 means do not draw any tic marks.

POLARTICSON ≠ 0 means draw tic marks.
