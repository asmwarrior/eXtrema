POLARPOWERAUTO

Syntax: SET POLARPOWERAUTO n
GET POLARPOWERAUTO { n }
Default: POLARPOWERAUTO = 1

POLARPOWERAUTO, along with POLARPOWER, controls the polar axis scale value that is appended to the polar axis text label. This scale value is needed when there are not enough digits allowed for the numbers labeling the polar axis.

POLARPOWERAUTO = 2 means determine POLARPOWER, but do not append the scale value to the text label, even if it is needed.

POLARPOWERAUTO = 1 means determine POLARPOWER, and append the scale value to the text label.

POLARPOWERAUTO = 0 means use the present value of POLARPOWER, and append the scale value to the text label.

If the user wishes to completely specify the appearance of the polar axis, POLARPOWERAUTO must be set to 0, otherwise the number of digits and decimal places, POLARNUMBEROFDIGITS and POLARNUMBEROFDECIMALS, may be changed.
