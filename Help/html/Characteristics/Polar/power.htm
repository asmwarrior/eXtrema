POLARPOWER

Syntax: SET POLARPOWER n
GET POLARPOWER { n }
Default: POLARPOWER = 0

POLARPOWER, along with POLARPOWERAUTO , controls the polar axis scale value that is appended to the polar axis text label. This scale value is a power of ten, that is, 10POLARPOWER, and the numbers labeling the polar axis should be multiplied by this scale value to get the correct graph units.
