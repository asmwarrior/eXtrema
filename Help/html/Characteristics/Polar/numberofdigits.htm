POLARNUMBEROFDIGITS

Syntax: SET POLARNUMBEROFDIGITS n
GET POLARNUMBEROFDIGITS { n }
Default: POLARNUMBEROFDIGITS = 5

POLARNUMBEROFDIGITS controls the total number of digits to be displayed in each of the numbers labeling the polar axis. If POLARNUMBEROFDIGITS is smaller than required to display the polar axis numbers, POLARNUMBEROFDIGITS will not be increased, but a scale value, 10n, will be appended to the polar axis text label. If POLARNUMBEROFDIGITS is larger than required, POLARNUMBEROFDIGITS will be reduced to the minimum value required.
