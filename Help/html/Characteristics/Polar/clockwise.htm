POLARCLOCKWISE

Syntax: SET POLARCLOCKWISE n
GET POLARCLOCKWISE { n }
Default value: POLARCLOCKWISE = 0

POLARCLOCKWISE controls whether or not the direction of increase of the angular coordinate is in clockwise or counter-clockwise.

POLARCLOCKWISE = 0 means the direction of increase of the angular coordinate is counter-clockwise.

POLARCLOCKWISE ≠ 0 means the direction of increase is clockwise.
