POLARLABELON

Syntax: SET POLARLABELON n
GET POLARLABELON { n }
Default: POLARLABELON = 0

POLARLABELON controls whether the automatic polar axis text label is drawn. The label itself is controlled with the POLARLABEL characteristic.

POLARLABELON = 0 → do not draw the automatic polar axis text label
POLARLABELON ≠ 0 → draw the automatic polar axis text label
