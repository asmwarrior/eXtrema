YLABEL

Syntax: SET YLABEL s
GET YLABEL { s }
Default: YLABEL is blank

Note: The value of the YLABEL characteristic is a string instead of a numeric value. So, if you enter: GET YLABEL X, the variable X will be a string variable.

The SET YLABEL command sets the automatic y-axis text label. Setting the label also turns it on. Use the SET YLABELON command to toggle off/on drawing the y-axis text label. The axis text labels are drawn only when the axes are drawn. The character string may contain format commands. Change the sizes of the text label with SET YLABELHEIGHT or SET %YLABELHEIGHT. The y-axis text label is drawn, centred, to the left of the y-axis.
