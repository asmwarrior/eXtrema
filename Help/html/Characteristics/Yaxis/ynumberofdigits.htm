YNUMBEROFDIGITS

Syntax: SET YNUMBEROFDIGITS n
GET YNUMBEROFDIGITS { n }
Default: YNUMBEROFDIGITS = 5

YNUMBEROFDIGITS controls the total number of digits to be displayed in each of the numbers labeling the y-axis.

If YNUMBEROFDIGITS is smaller than required to display the y-axis numbers, YNUMBEROFDIGITS will not be increased, but a scale value, 10n, will be appended to the y-axis text label.

If YNUMBEROFDIGITS is larger than required, YNUMBEROFDIGITS will be reduced to the minimum value required.
