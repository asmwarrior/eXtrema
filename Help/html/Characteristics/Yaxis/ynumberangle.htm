YNUMBERSANGLE

Syntax: SET YNUMBERSANGLE n
GET YNUMBERSANGLE { n }
Default: YNUMBERSANGLE = 0

YNUMBERSANGLE controls the angle of the base line for the numbers labeling the y-axis. YNUMBERSANGLE is the angle, in degrees, measured counterclockwise, between a line parallel to the x-axis and the base line of a number.
