YLARGETICLENGTH

Syntax: SET YLARGETICLENGTH n
SET %YLARGETICLENGTH n
GET YLARGETICLENGTH { n }
GET %YLARGETICLENGTH { n }
Default: %YLARGETICLENGTH = 2

YLARGETICLENGTH controls the length of the large, labeled, tic marks on the y-axis. There are two forms:

 YLARGETICLENGTH → the length, in world coordinate units, of the large tic marks on the y-axis
%YLARGETICLENGTH → the length, expressed as a percentage of the height of the graphics sub-window, that is, YLARGETICLENGTH = %YLARGETICLENGTH * (YUPPERWINDOW- YLOWERWINDOW)/100.
