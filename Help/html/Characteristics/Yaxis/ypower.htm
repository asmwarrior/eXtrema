YPOWER

Syntax: SET YPOWER n
GET YPOWER { n }
Default: YPOWER = 0

YPOWER, along with YPOWERAUTO , controls the y-axis scale value that is appended to the y-axis text label. This scale value is a power of ten, that is, 10YPOWER, and the numbers labeling the y-axis should be multiplied by this scale value to get the correct graph units.
