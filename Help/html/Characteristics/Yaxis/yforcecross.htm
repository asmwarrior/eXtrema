! YFORCECROSS

YFORCECROSS

Syntax: SET YFORCECROSS n
GET YFORCECROSS { n }
Default: YFORCECROSS = 0

YFORCECROSS controls where the x-axis is to cross the y-axis. The axes always cross at a large tic mark.

YFORCECROSS = 0 → the x-axis will cross the y-axis at YMIN
YFORCECROSS ≠ 0 → the x-axis will cross the y-axis at y = 0.0,
or at the large tic mark closest to y = 0.0
