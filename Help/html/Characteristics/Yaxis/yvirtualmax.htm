YVIRTUALMAX

Syntax: SET YVIRTUALMAX n
GET YVIRTUALMAX { n }
Default: YVIRTUALMAX = YMAX

YVIRTUALMAX controls the virtual maximum value for the y-axis. If YVIRTUALMAX is greater than YMAX, the actual y-axis maximum value will be YMAX, while the large, numbered, tic marks will end at YVIRTUALMAX, which may be 'off the end' of the y-axis. YVIRTUALMAX is always set to YMAX whenever YMAX is set, so you must set YVIRTUALMAX after YMAX.
