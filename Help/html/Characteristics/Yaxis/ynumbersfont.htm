YNUMBERSFONT

Syntax: SET YNUMBERSFONT s
GET YNUMBERSFONT { s }
Default: YLABELFONT = ARIAL

Note: The value of the YNUMBERSFONT characteristic is a string instead of a numeric value. So, if you enter: GET YNUMBERSFONT X , the variable X will be a string variable.

YNUMBERSFONT controls the font of the numbers labeling the y-axis. The parameter f should be the font name. If the font name is composed of more than one word, enclose the font name in quotes.

Examples:

SET YNUMBERSFONT ARIAL
SET YNUMBERSFONT 'ARIAL BOLD'
