YUPPERAXIS

Syntax: SET YUPPERAXIS n
SET %YUPPERAXIS n
GET YUPPERAXIS { n }
GET %YUPPERAXIS { n }
Default: %YUPPERAXIS = 90

YUPPERAXIS controls the position of the top end of the y-axis. This is also the vertical coordinate of the upper right hand corner of the graph box, if GRAPHBOX = 1. There are two forms:

 YUPPERAXIS → the vertical coordinate, in world coordinate units, of the top end of the y-axis
%YUPPERAXIS → the vertical coordinate, expressed as a percentage of the height of the graphics sub-window, that is, YUPPERAXIS = %YUPPERAXIS * (YUPPERWINDOW- YLOWERWINDOW)/100.
