YLABELON

Syntax: SET YLABELON n
GET YLABELON { n }
Default: YLABELON = 0

YLABELON controls whether the automatic y-axis text label is drawn. The label itself is changed with the YLABEL characteristic.

YLABELON = 0 → do not draw the automatic y-axis text label

YLABELON ≠ 0 → draw the automatic y-axis text label
