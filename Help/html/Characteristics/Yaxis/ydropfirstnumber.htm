YDROPFIRSTNUMBER

Syntax: SET YDROPFIRSTNUMBER n
GET YDROPFIRSTNUMBER { n }
Default: YDROPFIRSTNUMBER = 0

YDROPFIRSTNUMBER is another control on the numbering for the y-axis.

YDROPFIRSTNUMBER = 0 → the first, lowest value, numeric label will be displayed on the y-axis
YDROPFIRSTNUMBER ≠ 0 → the lowest value numeric label will not be displayed
