YIMAGTICANGLE

Syntax: SET YIMAGTICANGLE n
GET YIMAGTICANGLE { n }
Default: YIMAGTICANGLE = 90

YIMAGTICANGLE, along with YIMAGTICLENGTH , controls the location of the numbers labeling the y-axis at the large tic marks.

YIMAGTICANGLE is the angle, in degrees, measured counterclockwise, between the x-axis and a line joining the base of each large tic mark on the y-axis to the centre of the number labeling that tic mark.
