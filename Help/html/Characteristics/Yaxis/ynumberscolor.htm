YNUMBERSCOLOR

Syntax: SET YNUMBERSCOLOR n
SET YNUMBERSCOLOR colorname
GET YNUMBERSCOLOR { n }

Note: The GET YNUMBERSCOLOR command always returns the color index, and not the color name, even if YNUMBERSCOLOR was set with a color name.

SET YNUMBERSCOLOR controls the color of the numbers labeling the y-axis. The parameter c can be a numeric scalar or one of the fixed color names. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names.

For example:

SET YNUMBERSCOLOR 2 uses the colormap index 2
SET YNUMBERSCOLOR -2 uses the fixed color index 2
SET YNUMBERSCOLOR RED uses the fixed color name "RED"
