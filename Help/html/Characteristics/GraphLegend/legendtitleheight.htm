LEGENDTITLEHEIGHT

Syntax: SET LEGENDTITLEHEIGHT h
SET %LEGENDTITLEHEIGHT h
GET LEGENDTITLEHEIGHT { h }
GET %LEGENDTITLEHEIGHT { h }
Default value: %LEGENDTITLEHEIGHT = 3

LEGENDTITLEHEIGHT controls the height of the title to be drawn centred above the legend frame.

There are two ways to specify the height:

 LEGENDTITLEHEIGHT → specifies the height in world units
%LEGENDTITLEHEIGHT → specifies the height as a percentage of the height of the graphics sub-window, that is,
LEGENDTITLEHEIGHT = %LEGENDTITLEHEIGHT * (YUPPERWINDOW- YLOWERWINDOW)/100
