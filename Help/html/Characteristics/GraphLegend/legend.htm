LEGENDON

Syntax: SET LEGENDON n
GET LEGENDON { n }
Default value: LEGENDON = 0

LEGENDON controls whether or not a short line segment, with plotting symbols, is drawn along with a character string, when the GRAPH command is entered. The line segment is drawn with the same line type and plotting symbols as the data curve. If LEGENDON is on, a character string is expected as the first parameter of the GRAPH command.

LEGENDON = 0 → legend is off
LEGENDON ≠ 0 → legend is on
