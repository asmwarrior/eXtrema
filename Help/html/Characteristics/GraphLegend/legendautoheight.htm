LEGENDAUTOHEIGHT

Syntax: SET LEGENDAUTOHEIGHT n
GET LEGENDAUTOHEIGHT { n }
Default value: LEGENDAUTOHEIGHT = 1

LEGENDAUTOHEIGHT controls whether or not the height of the text portion of the graph legend entry is determined automatically so that it fits within the legend frame. If LEGENDAUTOHEIGHT is off, the height of each string will be the current value of TEXTHEIGHT .

LEGENDAUTOHEIGHT = 0 → off
LEGENDAUTOHEIGHT ≠ 0 → on
