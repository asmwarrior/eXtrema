XFORCECROSS

Syntax: SET XFORCECROSS n
GET XFORCECROSS { n }
Default value: XFORCECROSS = 0

XFORCECROSS controls where the y-axis is to cross the x-axis. The axes always cross at a large tic mark.

XFORCECROSS = 0 → the y-axis will cross the x-axis at XMIN
XFORCECROSS ≠ 0 → the y-axis will cross the x-axis at x = 0.0,
or at the large tic mark closest to x = 0.0
