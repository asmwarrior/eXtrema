XDROPFIRSTNUMBER

Syntax: SET XDROPFIRSTNUMBER n
GET XDROPFIRSTNUMBER { n }
Default value: XDROPFIRSTNUMBER = 0

XDROPFIRSTNUMBER is another control on the numbering for the x-axis.

XDROPFIRSTNUMBER = 0 → the first, lowest value, numeric label will be displayed on the x-axis
XDROPFIRSTNUMBER ≠ 0 → the lowest value numeric label will not be displayed
