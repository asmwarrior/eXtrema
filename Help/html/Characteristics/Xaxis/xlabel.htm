XLABEL

Syntax: SET XLABEL s
GET XLABEL { s }
Default value: XLABEL is blank

Note: The value of the XLABEL characteristic is a string instead of a numeric value. So, if you enter: GET XLABEL X, the variable X will be a string variable.

XLABEL is the automatic x-axis text label. Setting the label also turns it on. Use the XLABELON characteristic to toggle off/on drawing the x-axis text label. The axis text labels are drawn only when the axes are drawn. The character string may contain format commands. The size of the text label is controlled with XLABELHEIGHT or %XLABELHEIGHT. The x-axis text label is drawn, centred, below the x-axis.
