XAXISLINEWIDTH

Syntax: SET XAXISLINEWIDTH n
GET XAXISLINEWIDTH { n }
Default value: XAXISLINEWIDTH = 1

XAXISLINEWIDTH controls the line width of the x-axis.

The units of XAXISLINEWIDTH are 1/4 pts, where a pt is 1/72 of an inch. For example, a line width of 1 gives 1/288 inch wide lines, while a line width of 4 gives 1/72 inch wide lines.

The parameter n must be a scalar.
