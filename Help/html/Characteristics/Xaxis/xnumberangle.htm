XNUMBERSANGLE

Syntax: SET XNUMBERSANGLE n
GET XNUMBERSANGLE { n }
Default: XNUMBERSANGLE = 0

XNUMBERSANGLE controls the angle of the base line for the numbers labeling the x-axis. XNUMBERSANGLE is the angle, in degrees, measured counterclockwise, between a line parallel to the x-axis and the base line of a number.
