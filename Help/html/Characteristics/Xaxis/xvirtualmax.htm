XVIRTUALMAX

Syntax: SET XVIRTUALMAX n
GET XVIRTUALMAX { n }
Default: XVIRTUALMAX = XMAX

XVIRTUALMAX controls the virtual maximum value for the x-axis. If XVIRTUALMAX is greater than XMAX, the actual x-axis maximum value will be XMAX, while the large, numbered, tic marks will end at XVIRTUALMAX, which may be 'off the end' of the x-axis. XVIRTUALMAX is always set to XMAX whenever XMAX is set, so you must set XVIRTUALMAX after XMAX.
