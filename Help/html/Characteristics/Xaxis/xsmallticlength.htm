XSMALLTICLENGTH

Syntax: SET XSMALLTICLENGTH n
SET %XSMALLTICLENGTH n
GET XSMALLTICLENGTH { n }
GET %XSMALLTICLENGTH { n }
Default: %XSMALLTICLENGTH = 1

XSMALLTICLENGTH controls the length of the optional small tic marks on the x-axis. These are the unlabeled tic marks between the large, numbered, tic marks. There are two forms:

 XSMALLTICLENGTH → the length, in world coordinate units, of the small tic marks on the x-axis
%XSMALLTICLENGTH → the length, expressed as a percentage of the height of the graphics sub-window, that is, XSMALLTICLENGTH = %XSMALLTICLENGTH
* (YUPPERWINDOW - YLOWERWINDOW)/100.
