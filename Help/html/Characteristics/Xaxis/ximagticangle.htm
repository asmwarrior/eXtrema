XIMAGTICANGLE

Syntax: SET XIMAGTICANGLE a
GET XIMAGTICANGLE { n }
Default value: XIMAGTICANGLE = -90.0

XIMAGTICANGLE, along with XIMAGTICLENGTH , controls the location of the numbers labeling the x-axis at the large tic marks. XIMAGTICANGLE is the angle, in degrees, measured counterclockwise, between the x-axis and a line joining the base of each large tic mark on the x-axis to the centre of the number labeling that tic mark.
