NHISTORY

Syntax: SET NHISTORY n
GET NHISTORY { n }
Default: NHISTORY = -1

NHISTORY controls how many lines of history to display for each numeric variable as a result of the SHOW command.

NHISTORY < 0 → all stored history lines will be displayed
NHISTORY = 0,1 → only the latest history line will be displayed
NHISTORY > 0 → a maximum of NHISTORY lines of history will be displayed for each variable
