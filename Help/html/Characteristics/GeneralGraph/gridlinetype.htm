GRIDLINETYPE

Syntax: SET GRIDLINETYPE n
GET GRIDLINETYPE { n }
Default value: GRIDLINETYPE = 7

GRIDLINETYPE controls the line type of the grid lines drawn with the GRAPH command, either Cartesian or polar graphs.

The parameter n must be a scalar.

GRIDLINETYPE should be between 1 and 10, inclusive. See the figure below for examples of the line types. The length of the dashes in the dashed line types, the default types 3 to 10, are constant.
