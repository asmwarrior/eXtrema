ASPECTRATIO

Syntax: SET ASPECTRATIO n
GET ASPECTRATIO { n }
Default value: ASPECTRATIO = 8.5/11

ASPECTRATIO controls the aspect ratio of the visualization window, i.e., the height divided by the width.

The parameter n must be a scalar.

When drawing a polar plot, using the GRAPH\POLAR command, the drawing will look best if the aspect ratio is one.
