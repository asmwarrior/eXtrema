ARROWHEADLENGTH

Syntax: SET ARROWHEADLENGTH n
GET ARROWHEADLENGTH { n }
Default value: ARROWHEADLENGTH = 1.0

ARROWHEADLENGTH is a scale factor applied to the length of any arrow heads drawn with the FIGURE command or with the figure gui control on the visualization window speed bar. The default value is 1.0.

For example, to double the length of the arrow heads, use SET ARROWHEADLENGTH 2 .

The parameter n must be a scalar.
