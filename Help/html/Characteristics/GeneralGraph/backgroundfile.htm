BACKGROUNDFILE

Syntax: SET BACKGROUNDFILE pngfile
GET BACKGROUNDFILE { pngfile }
Default value: BACKGROUNDFILE = " "

Use SET BACKGROUNDFILE pngfile to change the background of the current window to a png image. The default for all windows is to have solid white background.

The parameter pngfile should be a png image file. The image will be resized to the proportions of the window.

Use SET BACKGROUNDCOLOR 0 to set the background color to white.

Use SET BACKGROUNDCOLOR to set the background for the current window to a solid color.
