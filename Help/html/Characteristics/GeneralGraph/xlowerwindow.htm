XLOWERWINDOW

Syntax: SET XLOWERWINDOW n
SET %XLOWERWINDOW n
GET XLOWERWINDOW { n }
GET %XLOWERWINDOW { n }
Default value: %XLOWERWINDOW = 0.0

XLOWERWINDOW determines the left edge of the graphics sub-window. There are two forms:

 XLOWERWINDOW → the left edge of the graphics sub-window, in world coordinate units
%XLOWERWINDOW → the left edge, expressed as a percentage of the width of the graphics world window
