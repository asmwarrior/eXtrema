YUPPERWINDOW

Syntax: SET YUPPERWINDOW n
SET %YUPPERWINDOW n
GET YUPPERWINDOW { n }
GET %YUPPERWINDOW { n }
Default value: %YUPPERWINDOW = 100.0

YUPPERWINDOW determines the top edge of the graphics sub-window. There are two forms:

 YUPPERWINDOW → the top edge of the graphics sub-window, in world coordinate units
%YUPPERWINDOW → the top edge, expressed as a percentage of the height of the graphics world window
