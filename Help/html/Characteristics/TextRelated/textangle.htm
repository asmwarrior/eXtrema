TEXTANGLE

Syntax: SET TEXTANGLE n
GET TEXTANGLE { n }
Default value: TEXTANGLE = 0

TEXTANGLE controls the angle of the text drawn with the TEXT command.

TEXTANGLE is the angle, in degrees, measured counterclockwise, between the base line of the character string and a horizontal line. The parameter a should be a numeric scalar.

The value of TEXTANGLE will be ignored if the value of TEXTALIGN is 10, 11, or 12.
