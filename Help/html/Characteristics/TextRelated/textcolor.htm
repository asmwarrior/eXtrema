TEXTCOLOR

Syntax: SET TEXTCOLOR n
SET TEXTCOLOR colorname
GET TEXTCOLOR { n }

Note: The GET TEXTCOLOR command always returns the color index, and not the color name, even if TEXTCOLOR was set with a color name.

TEXTCOLOR controls the color of the text string drawn with the TEXT command. The parameter c can be either a string or a numeric scalar. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names. See Colors

For example:

SET TEXTCOLOR 2 uses the colormap index 2
SET TEXTCOLOR -2 uses the fixed color index 2
SET TEXTCOLOR RED uses the fixed color name "RED"
