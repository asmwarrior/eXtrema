CURVELINETYPE

Syntax: SET CURVELINETYPE n
GET CURVELINETYPE { n }
Default value: CURVELINETYPE = 1

CURVELINETYPE controls the line type of the data curve drawn when the GRAPH command is entered. The value of CURVELINETYPE should be between 1 and 10, inclusive. See the figure below for examples of the line types. The length of the dashes in the dashed line types, the default types 3 to 10, are constant and do not depend on the separation between data points.
