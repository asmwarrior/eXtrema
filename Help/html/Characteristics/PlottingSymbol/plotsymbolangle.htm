PLOTSYMBOLANGLE

Syntax: SET PLOTSYMBOLANGLE n
GET PLOTSYMBOLANGLE { n }
Default value: PLOTSYMBOLANGLE = 0

PLOTSYMBOLANGLE controls the angle of the plotting symbols drawn when the GRAPH command is entered. PLOTSYMBOLANGLE is the angle, in degrees, measured counterclockwise, between the x-axis and a plotting symbol. The parameter n can be either a scalar or a vector.

If n is a scalar, every symbol will have the same angle.

If n is a vector, the ith symbol will have angle given by n[i]. In this case, the length of n must be the same as the length of the data vectors being plotted.
