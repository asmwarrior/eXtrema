PLOTSYMBOL

Syntax: SET PLOTSYMBOL n
GET PLOTSYMBOL { n }
Default: PLOTSYMBOL = 0

SET PLOTSYMBOL controls which plotting symbols to use when the GRAPH command is entered. The parameter n can be either a scalar or a vector.

If n is a scalar, every data point will have the same plotting symbol. If n is a vector, symbol n[i] will be drawn at data point (x[i],y[i]). In this case, the length of n must be the same as the lengths of x and y.

GET PLOTSYMBOL obtains the plotting symbol values or arrays, as set with the SET PLOTSYMBOL command. The type of output variable that will be made for each parameter depends on the type of variable used in the SET PLOTSYMBOL command.

Symbols

The symbols that will be plotted at the point (x[i],y[i]) are shown below.

s = 0 → no symbol at (x[i],y[i]), but connect that point to (x[i-1],y[i-1])
s > 0 → s connected to point (x[i-1],y[i-1])
s < 0 → s not connected to point (x[i-1],y[i-1])

There are 18 special plotting symbols. These symbols are centred at the data points.

The size of the plotting symbols can be changed with the SET PLOTSYMBOLSIZE command.
