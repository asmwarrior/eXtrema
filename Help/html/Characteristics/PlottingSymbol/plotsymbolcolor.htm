PLOTSYMBOLCOLOR

Syntax: SET PLOTSYMBOLCOLOR n
SET PLOTSYMBOLCOLOR colorname
GET PLOTSYMBOLCOLOR { n }
Default: PLOTSYMBOLCOLOR = black

Note: The GET PLOTSYMBOLCOLOR command always returns the color index, and not the color name, even if PLOTSYMBOLCOLOR was set with a color name.

SET PLOTSYMBOLCOLOR controls the color of the plotting symbols drawn when the GRAPH command is entered. The parameter c can be either a fixed color name, a numeric scalar, or a numeric vector. If c is a fixed color name or a numeric scalar, every symbol will have the same color. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names. If c is a numeric vector, the ith symbol's color will be c[i], where c[i] > 0 represents the index of the color in the currently loaded colormap, while c[i] < 0 represents the index of the fixed color. In the vector case, the length of c must be the same as the length of the data vectors entered with the GRAPH command.

For example:

SET PLOTSYMBOLCOLOR 2 uses the colormap index 2
SET PLOTSYMBOLCOLOR -2 uses the fixed color index 2
SET PLOTSYMBOLCOLOR RED uses the fixed color name "RED"
