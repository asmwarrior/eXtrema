STACKEXTENSION

Syntax: SET STACKEXTENSION s
GET STACKEXTENSION { s }
Default value: STACKEXTENSION = stk

Note: The value of the STACKEXTENSION characteristic is a string and not a numeric value. So, if you enter: GET STACKEXTENSION X, the variable X will be a string variable.

The SET STACKEXTENSION command is used to redefine the default file extension for stack files. The original default file extension is stk. This file extension applies to the STACK command only. If you enter a file name without a file extension to the STACK command, the default extension will be automatically appended to the file name.

The case of the extension is kept as entered.

Examples

If you enter the command STACK some_file, the STACKEXTENSION will be automatically appended to the file name, so, unless you have changed the STACKEXTENSION with the SET STACKEXTENSION command, your stack file will be some_file.stk. If you enter the command STACK some_file.xxx, your stack file name will be left as some_file.xxx.

If you have a stack file named stack_file.stk, you can execute this file with the command
@stack_file or with @stack_file.stk
