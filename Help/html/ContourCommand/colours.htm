Contour level color

Syntax: CONTOUR\COLORS colr { x y } v nctr { min { incr }}
CONTOUR\INTERP\COLORS colr ntrp { x y } v num { min { incr }}
CONTOUR\SPECIFIC\COLORS colr { x y } v lvls
CONTOUR\SPECIFIC\INTERP\COLORS colr ntrp { x y } v lvls

Contours in color can be obtained using the \COLOR qualifier. A color vector, colr, is expected as the first parameter. The length of colr should be the same as the number of contours requested. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names.
