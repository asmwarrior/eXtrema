Contour labels

By default, contours are labeled with the actual contour level, using three significant digits. See the discussion on the \LEGEND qualifier for an alternate contour labeling facility.

Contour label size

The size of the contour labels is CONTOURLABELHEIGHT. If labels are not desired on the contours, use the SET CONTOURLABELHEIGHT 0 command to set the label height to zero before entering the CONTOUR command.

Contour label separation

The separation between contour labels is controlled with CONTOURLABELSEPARATION or %CONTOURLABELSEPARATION, which can be changed with the SET CONTOURLABELSEPARATION command. If %CONTOURLABELSEPARATION is set, the separation is a percentage of the height of the window, that is, YUPPERWINDOW-YLOWERWINDOW. If CONTOURLABELSEPARATION is set, the separation is expressed in inches or centimeters, depending on the units. The default is CONTOURLABELSEPARATION = 10.
