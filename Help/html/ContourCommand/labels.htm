Contour labels

By default, contours are labeled with the actual contour level, using three significant digits. See the discussion on the \LEGEND qualifier for an alternate contour labeling facility.

Contour label size

The size of the contour labels is CONTOURLABELHEIGHT. If labels are not desired on the contours, use the SET CONTOURLABELHEIGHT 0 command to set the label height to zero before entering the CONTOUR command.
