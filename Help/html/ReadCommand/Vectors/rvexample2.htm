Example 2

Suppose ASCII file DUM.DAT is as follows: 
1  23.7   0.1000E-5
2 -31.4   0.2000E-3
3  9.09   0.3000E-1
4  10.001 0.4000E+1
5  -2.0   0.5000E+2
6  30.2   0.6000E+3

READ DUM.DAT X Y

results in: X = [ 1; 2; 3; 4; 5; 6 ]
Y = [ 23.7; -31.4; 9.09; 10.001; -2.0; 30.2]

READ DUM.DAT\3 X Y

results in: X = [ 3; 4; 5; 6 ]
Y = [ 9.09; 10.001; -2.0; 30.2 ]

READ DUM.DAT\[2:4] X/3 Y

results in: X = [ .0002; .03; 4.0 ]
Y = [ -31.4; 9.09; 10.001 ]
