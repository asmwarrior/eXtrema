Write a matrix

Syntax: WRITE\MATRIX file m
Qualifiers: \APPEND
Examples: WRITE\MATRIX FILE.DAT M
WRITE\MATRIX\APPEND FILE.DAT M

The WRITE\MATRIX command writes a matrix to a file. If \APPEND is used, and if the output file already exists, the data will be appended onto the end of the file.
