Mean

Syntax: s = MEAN(x)
s = MEAN(w,x)

The MEAN function only accepts one or two vector arguments. If only one vector is entered, the function returns the mean, or average, of that vector's data. If two vectors are entered, the first vector holds the weights and the second vector holds the data. The result is always a scalar value.
