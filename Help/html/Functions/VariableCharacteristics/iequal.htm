IEQUAL

Syntax: scalar = IEQUAL(vector,scalar)

The IEQUAL function returns the index of vector which corresponds to where vector is equal to the value of scalar, that is, IEQUAL returns the first j where vector[j] = scalar. If the value of scalar is not in vector then IEQUAL returns zero (0).

Example

Suppose you have vector X = [1;2;3;4;5]. If you enter A=IEQUAL(X,2), the scalar A will have the value 2.
