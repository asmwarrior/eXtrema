CANOPEN

Syntax: scalar = CANOPEN(filename)

The CANOPEN function accepts a string as argument, which is interpreted as a file name. If filename exists, the function returns the value one (1). If filename does not exist, the function returns the value zero (0). The filename string argument will not be modified.

Examples

TEST = CANOPEN('/home/user/foo.dat')

or

FILENAME = '/home/user/foo.dat'
TEST = CANOPEN(FILENAME)
