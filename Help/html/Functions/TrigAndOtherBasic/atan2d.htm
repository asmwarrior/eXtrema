ATAN2D

Syntax: vout = ATAN2D( v1, v2 )

The ATAN2D(v1,v2) function returns the Arc Tangent of v1/v2 in degrees, with -180° < ATAN2D(v1,v2) ≤ 180°.

  • If v1 > 0, the result is positive.
  • If v1 = 0, the result is zero if v2 > 0 and 180° if v2 < 0.
  • If v1 < 0, the result is negative. If the value of the second argument is zero, the absolute value of the result is 90°.

Both arguments must not have the value zero.
