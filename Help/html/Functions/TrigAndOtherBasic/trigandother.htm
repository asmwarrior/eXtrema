SIGN(x,y) transfer of sign

All of these functions expect numeric arguments, and operate on an element by element basis. The trigonometric and other basic functions will always have a resultant type which will be the same as the type of its argument, that is, a scalar argument results in a scalar, a vector argument results in a vector with the same length as the argument, a matrix (tensor) argument results in a matrix (tensor) with the same size and shape as the argument. Some of these element by element functions, such as RAN, ATAN2, or MIN do require some explanation, but the definitions of most are assumed to be obvious to the reader.
