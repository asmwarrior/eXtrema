WORLD command

Syntax: WORLD vxin vyin { vxout vyout }
Qualifiers: \PERCENT
Defaults: absolute coordinates returned

The WORLD command converts graph coordinates into world coordinates (centimeters), or into percentages if the \PERCENT qualifier is used.

The variables vxin and vyin may be scalars or vectors, but they must both be the same type. The resultant values are displayed on the monitor screen. If output variables vxout and vyout are present, the values are stored there. The type of variable created depends on the variable type of vxin and vyin.
