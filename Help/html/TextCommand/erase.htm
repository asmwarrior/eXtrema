ERASE

The text strings entered with the TEXT command are stored in a FILO (first in last out) stack. If TEXT\ERASE is entered, the last text string drawn will be erased, and popped off the stack. If the TEXT\ERASE command is issued again, the next to last text string drawn will be erased, and popped off the stack. And so on, until the stack is empty.
