SURFACE command

Syntax: SURFACE matrix { xangle { zangle }}
Qualifiers: \COLOR, \HISTOGRAM
Defaults: xangle = 40 degrees, yangle = 40 degrees,
\-COLOR, \-HISTOGRAM
Examples: SURFACE M
SURFACE\COLOR M
SURFACE\HIST M 30 60

This command displays a matrix in the form of a 3-d figure which can be rotated in space. The appearance of the figure is controlled by qualifiers and optional parameters.

The \HISTOGRAM qualifier causes the surface plot to be "lego" type.

The \COLOR qualifier causes the surface plot to be in color. The range of the data is divided into 10 equal regions, and each is drawn in a different color. This allows the user to quickly identify areas of similar height.

The optional parameter xangle is the rotation angle (in degrees) about the x-axis. The default is 40 degrees. The optional parameter zangle is the rotation angle (in degrees) about the z-axis. The default is 40 degrees.
