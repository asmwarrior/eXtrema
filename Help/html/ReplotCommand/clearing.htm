Clearing the graphics

Use the CLEAR\-REPLOT command to clear the graphics page without emptying the replot buffers.

Use the CLEAR command with no qualifiers and no parameters to clear the graphics page and empty the replot buffers.

Use the CLEAR n command to erase the graphics from window number n and empty the replot buffers.

Use the CLEAR\-REPLOT n command to erase the graphics from window number n, but leave the replot buffers.
