LEGENDTITLECOLOR

Syntax: SET LEGENDTITLECOLOR c

LEGENDTITLECOLOR controls the color of the title drawn centred above the graph legend frame. The parameter c can be either a string or a numeric scalar. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names. See Colors

For example:

SET LEGENDTITLECOLOR 2 uses the colormap index 2
SET LEGENDTITLECOLOR -2 uses the fixed color index 2
SET LEGENDTITLECOLOR RED uses the fixed color name "RED"
