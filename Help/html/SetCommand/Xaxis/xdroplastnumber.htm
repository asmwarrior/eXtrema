XDROPLASTNUMBER

Syntax: SET XDROPLASTNUMBER n
Default value: XDROPLASTNUMBER = 0

XDROPLASTNUMBER is another control on the numbering for the x-axis.

XDROPLASTNUMBER = 0 → the last, highest value, numeric label will be displayed on the x-axis
XDROPLASTNUMBER ≠ 0 → the highest value numeric label will not be displayed
