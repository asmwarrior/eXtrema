XPOWERAUTO

Syntax: SET XPOWERAUTO a
Default value: XPOWERAUTO = 1

XPOWERAUTO, along with XPOWER, controls the x-axis scale value that is appended to the x-axis text label. This scale value is needed when there are not enough digits allowed for the numbers labeling the x-axis.

XPOWERAUTO
2 → determine XPOWER, but do not append the scale value to the text label, even if it is needed
1 → determine XPOWER, and append the scale value to the text label
0 → use the present value of XPOWER, and append the scale value to the text label

If the user wishes to completely specify the appearance of the x-axis, XPOWERAUTO must be set to 0, otherwise the number of digits and decimal places, XNUMBEROFDIGITS and XNUMBEROFDECIMALS, may be changed.
