XLABEL

Syntax: SET XLABEL `string'
Default value: XLABEL = ` '

The SET XLABEL command sets the automatic x-axis text label. Use the SET XLABELON command to toggle off/on drawing the x-axis text label. The axis text labels are drawn only when the axes are drawn. The character string may contain format commands. Change the sizes of the text label with SET XLABELHEIGHT or SET %XLABELHEIGHT. The x-axis text label is drawn, centred, below the x-axis.
