XLEADINGZEROS

Syntax: SET XLEADINGZEROS n
Default value: XLEADINGZEROS = 0

XLEADINGZEROS controls whether leading zeros are displayed on the x-axis numbers.

XLEADINGZEROS = 0 → numbers displayed on the x-axis will not have leading zeros if they are between 0 and 1
XLEADINGZEROS ≠ 0 → numbers displayed on the x-axis will have leading zeros if they are between 0 and 1

The default is to not display these leading zeros. Numbers that are between -1 and 0 always have the leading zero displayed.
