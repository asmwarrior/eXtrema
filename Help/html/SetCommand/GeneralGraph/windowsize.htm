WINDOWSIZE

Syntax: SET WINDOWSIZE s

Use WINDOWSIZE to change the size of the main graphics window by the factor entered. For example, SET WINDOWSIZE 1.2 increases the window size by 20%, while SET WINDOWSIZE 0.2 decreases the size by 80%.

SET WINDOWSIZE 0 resets the window size to the original default size.
