XUPPERWINDOW

Syntax: SET XUPPERWINDOW s
SET %XUPPERWINDOW s
Default value: %XUPPERWINDOW = 100.0

XUPPERWINDOW determines the right edge of the graphics sub-window. There are two forms:

 XUPPERWINDOW → the right edge of the graphics sub-window, in world coordinate units
%XUPPERWINDOW → the right edge, expressed as a percentage of the width of the graphics world window
