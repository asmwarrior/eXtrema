YLOWERWINDOW

Syntax: SET YLOWERWINDOW s
SET %YLOWERWINDOW s
Default value: %YLOWERWINDOW = 0.0

YLOWERWINDOW determines the bottom edge of the graphics sub-window. There are two forms:

 YLOWERWINDOW → the bottom edge of the graphics sub-window, in world coordinate units
%YLOWERWINDOW → the bottom edge, expressed as a percentage of the height of the graphics world window
