COLOR

Syntax: SET COLOR c

SET COLOR is a shorthand way to set TEXTCOLOR, XAXISCOLOR, YAXISCOLOR, PLOTSYMBOLCOLOR, LINECOLOR, XLABELCOLOR, YLABELCOLOR, XNUMBERSCOLOR, YNUMBERSCOLOR, LEGENDTITLECOLOR, and FILLCOLOR, all to the same color. The parameter c can be either a string or a numeric scalar. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names. See Colors.

For example:

SET COLOR 2 uses the colormap index 2
SET COLOR -2 uses the fixed color index 2
SET COLOR RED uses the fixed color name "RED"
