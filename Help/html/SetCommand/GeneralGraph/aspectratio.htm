ASPECTRATIO

Syntax: SET ASPECTRATIO r

The aspect ratio of the graphics page is the height of the page divided by the width of the page.

Use SET ASPECTRATIO to change the aspect ratio of the graphics page. For example, SET ASPECTRATIO 8.5/11 sets the aspect ratio to be typical letter size in landscape orientation, while SET ASPECTRATIO 11/8.5 sets the aspect ratio to be letter size in portrait orientation.
