YLABELCOLOR

Syntax: SET YLABELCOLOR c

SET YLABELCOLOR controls the color of the automatic y-axis text label. The parameter c can be a numeric scalar or one of the fixed color names. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names.

For example:

SET YLABELCOLOR 2 uses the colormap index 2
SET YLABELCOLOR -2 uses the fixed color index 2
SET YLABELCOLOR RED uses the fixed color name "RED"
