YMIN

Syntax: SET YMIN n
Default value: YMIN = 0.0

YMIN controls the minimum value to be displayed on the y-axis.

If |YLOGBASE| > 1.0, then YMIN is assumed to be the exponent for the minimum value on the y-axis. The minimum value displayed on the y-axis is |YLOGBASE|YMIN . Integer exponents are always used for the axis numbering, so the y-axis may not begin on a large tic mark. If |YLOGBASE| ≤ 1.0, YMIN is the actual minimum value for the y-axis, and the y-axis always begins on a large tic mark.
