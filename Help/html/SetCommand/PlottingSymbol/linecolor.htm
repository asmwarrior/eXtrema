CURVECOLOR

Syntax: SET CURVECOLOR c

SET CURVECOLOR controls the color of the data curve drawn when the GRAPH command is entered. The parameter c can be a numeric scalar or one of the fixed color names.

For example:

SET CURVECOLOR 2 uses the colormap index 2
SET CURVECOLOR RED uses the fixed color name "RED"
