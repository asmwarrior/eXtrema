PLOTSYMBOLANGLE

Syntax: SET PLOTSYMBOLANGLE a
Default value: PLOTSYMBOLANGLE = 0

SET PLOTSYMBOLANGLE controls the angle of the plotting symbols drawn when the GRAPH command is entered. PLOTSYMBOLANGLE is the angle, in degrees, measured counterclockwise, between the x-axis and a plotting symbol. The parameter a can be either a scalar or a vector. If a is a scalar, every symbol will have the same angle. If a is a vector, the ith symbol will have angle given by a[i]. In this case, the length of a must be the same as the length of the data vectors entered with the GRAPH command.
