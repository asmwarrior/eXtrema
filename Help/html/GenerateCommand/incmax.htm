Increment and maximum

Syntax: GENERATE x min inc max

If the increment and the maximum value are given, the number of points will be ignored. The formula: x[i] = min+(i-1)*inc    for    i=[1;2;3;...] will be applied until the next value would be greater than the specified maximum. If the calculated maximum is different than the given maximum, a warning message will be displayed. The calculated maximum will be the last value stored in the vector.

For example, suppose you enter the command: GENERATE X -1 .41 1 then we have X = [-1;-0.59;-0.18;0.23;0.64]. The warning message:

       GENERATE warning: calculated maximum = 0.64000000E+00
                              given maximum = 0.10000000E+01
will be displayed.
