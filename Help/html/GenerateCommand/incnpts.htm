Increment and number of points

Syntax: GENERATE x min inc ,, npts

If the increment and the number of points are given, the formula: x[i] = min+(i-1)*inc    for    1≤i≤npts is applied directly.

For example, after the command: GENERATE X -1 .5 ,, 4 we have X = [-1; -0.5; 0; 0.5]
