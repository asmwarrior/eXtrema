Average

Syntax: BIN\AVERAGE x xbin xcount
BIN\AVERAGE\EDGES x xbin xcount

The \AVERAGE qualifier means that the output xcount vector will contain the average value for each bin. An internal counter is kept for each bin, and the value for xcount[i] will be divided by the number of events in bin i before it is output. \AVERAGE can be used with \WEIGHTS and \EDGES, but not with \EMPTY.
