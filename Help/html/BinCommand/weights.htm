Weights

Syntax: BIN\WEIGHTS w x xbin xcount
BIN\AVERAGE\WEIGHTS w x xbin xcount
BIN\EMPTY\WEIGHTS w x xbin xcount
BIN\EDGES\WEIGHTS w x xbin xcount
BIN\EDGES\EMPTY\WEIGHTS w x xbin xcount
BIN\EDGES\EMPTY\AVERAGE\WEIGHTS w x xbin xcount

By default, a bin count is incremented by 1 for every event that goes in a bin. If a weight vector is entered, you must indicate that it is there by using \WEIGHTS qualifier. The weight w must be a vector with the same length as x. The ith event causes the bin count to be incremented by w[i].
