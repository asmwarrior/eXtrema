Examples

DESTROY\MATRICES X[5:25:5] Y S[2]

will destroy all matrices; elements 5, 10, 15, 20, and 25 from vector X; the variable Y; the second string from string array variable S. The variables X and S will be compressed.

DESTROY X[WHERE((X>M)&(X<N))]

will delete from a vector all elements that have a value between M and N, assuming that M and N are scalars.
