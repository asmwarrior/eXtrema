GRAPH command

Syntax: GRAPH { 'legendtext' } x y { ye1 { xe1 { ye2 { xe2 }}}}
Qualifiers: \AXESONLY, \OVERLAY, \POLAR, \REPLOT, \HISTOGRAM, \YONRIGHT, \XONTOP, \SMOOTH
Defaults: axes drawn, \REPLOT, \-POLAR, \-HISTOGRAM, \-YONRIGHT, \-XONTOP, \-SMOOTH,   legendtext ignored
Examples: GRAPH X Y
GRAPH 'legend entry' X Y YERR XERR
GRAPH\OVERLAY X Y
GRAPH\OVERLAY\SMOOTH X Y
GRAPH\HISTOGRAM X Y
GRAPH\POLAR RAD THETA

command
The GRAPH command draws: data with axes GRAPH
just the data GRAPH\OVERLAY
just the axes GRAPH\AXESONLY
y-axis on the right GRAPH\YONRIGHT
x-axis on the top GRAPH\XONTOP
smooth curve through the data GRAPH\SMOOTH

The data curve may be a histogram. The parameters must be vectors, but can be vectors of length one. The input vectors can have different lengths as the minimum length of all of the input vectors will be used.
