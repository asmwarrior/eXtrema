POLARDROPFIRSTNUMBER

Syntax: SET POLARDROPFIRSTNUMBER n
GET POLARDROPFIRSTNUMBER { n }
Default value: POLARDROPFIRSTNUMBER = 0

POLARDROPFIRSTNUMBER is another control on the numbering for the polar axis.

If POLARDROPFIRSTNUMBER = 0, the first, lowest value, numeric label will be displayed on the polar axis

If POLARDROPFIRSTNUMBER ≠ 0, the lowest value numeric label will not be displayed.
