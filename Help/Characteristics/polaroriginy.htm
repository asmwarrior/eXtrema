POLARORIGINY

Syntax: SET POLARORIGINY n
SET %POLARORIGINY n
GET POLARORIGINY { n }
GET %POLARORIGINY { n }
Default: %POLARORIGINY = 50

POLARORIGINY controls the vertical coordinate of the polar plot origin. There are two forms:

 POLARORIGINY → the vertical coordinate, in world coordinate units, of the origin of the polar plot
%POLARORIGINY → the vertical coordinate, expressed as a percentage of the height of the graphics sub-window, that is, POLARORIGINY = %POLARORIGINY * (YUPPERWINDOW- YLOWERWINDOW)/100.
