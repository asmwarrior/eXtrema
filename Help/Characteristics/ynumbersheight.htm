YNUMBERSHEIGHT

Syntax: SET YNUMBERSHEIGHT n
SET %YNUMBERSHEIGHT n
GET YNUMBERSHEIGHT { n }
GET %YNUMBERSHEIGHT { n }
Default: %YNUMBERSHEIGHT = 2

YNUMBERSHEIGHT controls the size of the numbers labeling the y-axis. There are two forms:

 YNUMBERSHEIGHT → the height, in world coordinate units, of the numbers labeling the y-axis
%YNUMBERSHEIGHT → the height, expressed as a percentage of the height of the graphics sub-window, that is, YNUMBERSHEIGHT = %YNUMBERSHEIGHT
* (YUPPERWINDOW - YLOWERWINDOW)/100.
