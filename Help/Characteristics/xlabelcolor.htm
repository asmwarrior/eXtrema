XLABELCOLOR

Syntax: SET XLABELCOLOR n
SET XLABELCOLOR colorname
GET XLABELCOLOR { n }
Default: XLABELCOLOR = black

Note: The GET XLABELCOLOR command always returns the color index, and not the color name, even if XLABELCOLOR was set with a color name.

XLABELCOLOR controls the color of the automatic x-axis text label. The parameter c can be a numeric scalar or one of the fixed color names. The parameter c can be either a string or a numeric scalar. A positive numeric value represents the index of the color in the currently loaded colormap. A negative number represents the index of the fixed color. A string value should be one of the fixed color names.

For example:

SET XLABELCOLOR 2 uses the colormap index 2
SET XLABELCOLOR -2 uses the fixed color index 2
SET XLABELCOLOR RED uses the fixed color name "RED"
