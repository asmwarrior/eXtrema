ERRORFILL

Syntax: SET ERRORFILL n
GET ERRORFILL { n }
Default value: ERRORFILL = 0

If the \ERRORFILL qualifier is used when reading in vectors, an invalid field in the data file causes either the entire record to be filled with the value of ERRORFILL if a format is used, or only that invalid field will be filled with the value of ERRORFILL if no format is used.
