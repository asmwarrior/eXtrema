CURVECOLOR

Syntax: SET CURVECOLOR n
SET CURVECOLOR colorname
GET CURVECOLOR { n }
Default: CURVECOLOR = black

Note: The GET CURVECOLOR command always returns the color index, and not the color name, even if CURVECOLOR was set with a color name.

CURVECOLOR controls the color of the data curve drawn when the GRAPH command is entered. The parameter can be a numeric scalar or one of the fixed color names.

For example:

SET CURVECOLOR 2 uses the colormap index 2
SET CURVECOLOR RED uses the fixed color name "RED"
