CONTOURLABELSEPARATION

Syntax: SET CONTOURLABELSEPARATION n
SET %CONTOURLABELSEPARATION n
GET CONTOURLABELSEPARATION { n }
GET %CONTOURLABELSEPARATION { n }
Default value: CONTOURLABELSEPARATION = 10

CONTOURLABELSEPARATION controls the separation distance between text labels drawn as contour labels when using the CONTOUR command.

There are two ways to specify the separation:

 CONTOURLABELSEPARATION → specifies the height in world units
%CONTOURLABELSEPARATION → specifies the height as a percentage of the height of the graphics sub-window, that is,
CONTOURLABELHEIGHT = %CONTOURLABELHEIGHT * (YUPPERWINDOW-YLOWERWINDOW)/100
