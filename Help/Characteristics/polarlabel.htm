POLARLABEL

Syntax: SET POLARLABEL s
GET POLARLABEL { s }
Default value: POLARLABEL is blank

Note: The value of the POLARLABEL characteristic is a string instead of a numeric value. So, if you enter: GET POLARLABEL X, the variable X will be a string variable.

The SET POLARLABEL command sets the automatic polar axis text label. Setting the label also sets POLARLABELON to 1 (meaning true).

The axis text label is drawn only when the axes are drawn. The axis text label is drawn, centred, below the polar axis.

The character string may contain format commands.

The size of the text label is controlled by the POLARLABELHEIGHT characteristic.
