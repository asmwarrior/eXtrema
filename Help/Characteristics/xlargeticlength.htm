XLARGETICLENGTH

Syntax: SET XLARGETICLENGTH n
SET %XLARGETICLENGTH n
GET XLARGETICLENGTH { n }
GET %XLARGETICLENGTH { n }
Default: %XLARGETICLENGTH = 2

XLARGETICLENGTH controls the length of the large, labeled, tic marks on the x-axis. There are two forms:

 XLARGETICLENGTH → the length, in world coordinate units, of the large tic marks on the x-axis
%XLARGETICLENGTH → the length, expressed as a percentage of the height of the graphics sub-window, that is, XLARGETICLENGTH = %XLARGETICLENGTH
* (YUPPERWINDOW - YLOWERWINDOW)/100.
