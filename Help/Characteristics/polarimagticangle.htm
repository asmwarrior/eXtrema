POLARIMAGTICANGLE

Syntax: SET POLARIMAGTICANGLE a
GET POLARIMAGTICANGLE { n }
Default value: POLARIMAGTICANGLE = -90.0

POLARIMAGTICANGLE, along with POLARIMAGTICLENGTH, controls the location of the numbers labeling the polar axis at the large tic marks. POLARIMAGTICANGLE is the angle, in degrees, measured counterclockwise, between the polar axis and a line joining the base of each large tic mark on the polar axis to the centre of the number labeling that tic mark.
