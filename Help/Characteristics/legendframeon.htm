LEGENDFRAMEON

Syntax: SET LEGENDFRAMEON n
GET LEGENDFRAMEON { n }
Default value: LEGENDFRAMEON = 1

LEGENDFRAMEON controls whether or not a frame is drawn around the graph legend.

The size and location of the frame is controlled with the LEGENDFRAME characteristic.

LEGENDFRAMEON = 0 → no frame is drawn
LEGENDFRAMEON ≠ 0 → a frame will be drawn
