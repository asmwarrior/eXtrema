POLARLEADINGZEROS

Syntax: SET POLARLEADINGZEROS n
GET POLARLEADINGZEROS { n }
Default: POLARLEADINGZEROS = 0

POLARLEADINGZEROS controls whether leading zeros are displayed on the polar axis numbers.

If POLARLEADINGZEROS = 0, numbers displayed on the polar axis will not have leading zeros if they are between 0 and 1

If POLARLEADINGZEROS ≠ 0, numbers displayed on the polar axis will have leading zeros if they are between 0 and 1

The default is to not display these leading zeros. Numbers that are between -1 and 0 always have the leading zero displayed.
