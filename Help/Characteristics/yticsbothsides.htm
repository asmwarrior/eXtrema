YTICSBOTHSIDES

Syntax: SET YTICSBOTHSIDES n
GET YTICSBOTHSIDES { n }
Default: YTICSBOTHSIDES = 0

YTICSBOTHSIDES controls whether or not tic marks, both large and small, are drawn on both sides of the y-axis.

YTICSBOTHSIDES = 0 → only draw tic marks on one side of the axis line. YTICANGLE controls the side on which the tic marks will appear
YTICSBOTHSIDES ≠ 0 → draw tic marks on both sides of the axis line
