POLAROFFSET

Syntax: SET POLAROFFSET n
GET POLAROFFSET { n }
Default: POLAROFFSET = 0

POLAROFFSET is another control on the numbering for the polar axis.

If |POLARMOD| > 1, then POLAROFFSET is added to the numbers to be displayed on the polar axis. If |POLARMOD| ≤ 1, then POLAROFFSET is ignored.
