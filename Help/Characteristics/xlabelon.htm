XLABELON

Syntax: SET XLABELON n
GET XLABELON { n }
Default: XLABELON = 0

XLABELON controls whether the automatic x-axis text label is drawn. The label itself is controlled with the XLABEL characteristic.

XLABELON = 0 → do not draw the automatic x-axis text label
XLABELON ≠ 0 → draw the automatic x-axis text label
