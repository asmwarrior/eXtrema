XLABELFONT

Syntax: SET XLABELFONT s
GET XLABELFONT { s }

Note: The value of the XLABELFONT characteristic is a string instead of a numeric value. So, if you enter: GET XLABELFONT X , the variable X will be a string variable.

XLABELFONT controls the font of the automatic x-axis text label. The parameter f should be the font name. If the font name is composed of more than one word, enclose the font name in quotes.

Examples:

SET XLABELFONT ARIAL
SET XLABELFONT 'ARIAL BOLD'
