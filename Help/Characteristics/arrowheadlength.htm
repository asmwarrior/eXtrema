ARROWHEADLENGTH

Syntax: SET ARROWHEADLENGTH n
GET ARROWHEADLENGTH { n }
Default value: 1.0

ARROWHEADLENGTH is a scale factor applied to the length of any arrow heads drawn with the FIGURE command. The default value is 1.0.

The parameter n must be a scalar.

For example, to double the length of the arrow heads, use SET ARROWHEADLENGTH 2 .
