POLARCOMPASSLABELS

Syntax: SET POLARCOMPASSLABELS n
GET POLARCOMPASSLABELS { n }
Default: POLARCOMPASSLABELS = 0

POLARCOMPASSLABELS controls whether or not each of the radial polar axes are labeled with the compass direction abbreviations or with the angle of that axis.

POLARCOMPASSLABELS = 0 means label the outer end of each axis with the angle of that axis relative to the 0° radial axis.

POLARCOMPASSLABELS ≠ 0 means label the outer end of each axis with the abbreviation for the compass direction. This only applies if POLARNAXES is one of 1, 2, 4, 8, or 16.

The height of these labels is controlled by the POLARLABELHEIGHT characteristic.
