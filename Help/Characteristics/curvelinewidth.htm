CURVELINEWIDTH

Syntax: SET CURVELINEWIDTH n
GET CURVELINEWIDTH { n }
Default value: CURVELINEWIDTH = 1

CURVELINEWIDTH controls the line width of the data curve drawn when the GRAPH command is entered. The units of LINEWIDTH are 1/4 pts, where a pt is 1/72 of an inch. For example, a line width of 1 gives 1/288 inch wide lines, while a line width of 4 gives 1/72 inch wide lines.

The parameter n must be a scalar value.
