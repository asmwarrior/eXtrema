LEGENDENTRYLINEON

Syntax: SET LEGENDENTRYLINEON n
GET LEGENDENTRYLINEON { n }
Default value: LEGENDENTRYLINEON = 1

LEGENDENTRYLINEON controls whether or not a line segment is drawn connecting the plot symbols as part of the legend entry.

LEGENDENTRYLINEON = 0 → no line is drawn
LEGENDENTRYLINEON ≠ 0 → a line will be drawn
