YLOGSTYLE

Syntax: SET YLOGSTYLE n
GET YLOGSTYLE { n }
Default: YLOGSTYLE = 1

YLOGSTYLE controls whether or not the numbers labeling the large tic marks on the y-axis are displayed as powers of YLOGBASE when YLOGBASE > 1.0.

YLOGSTYLE = 0 → display the numbers in decimal form
YLOGSTYLE ≠ 0 → display the numbers in exponential form
