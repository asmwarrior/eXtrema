POLARSMALLTICLENGTH

Syntax: SET POLARSMALLTICLENGTH n
SET %POLARSMALLTICLENGTH n
GET POLARSMALLTICLENGTH { n }
GET %POLARSMALLTICLENGTH { n }
Default: %POLARSMALLTICLENGTH = 0.5

POLARSMALLTICLENGTH controls the length of the optional small tic marks on the polar axis. These are the unlabeled tic marks between the large, numbered, tic marks. There are two forms:

 POLARSMALLTICLENGTH → the length, in world coordinate units, of the small tic marks on the polar axis
%POLARSMALLTICLENGTH → the length, expressed as a percentage of the height of the graphics sub-window, that is, POLARSMALLTICLENGTH = %POLARSMALLTICLENGTH * (YUPPERWINDOW- YLOWERWINDOW)/100.
