LEGENDTITLE

Syntax: SET LEGENDTITLE s
GET LEGENDTITLE { s }
Default value: LEGENDTITLE is blank

Note: The value of the LEGENDTITLE characteristic is a string and not a numeric value. So, if you enter: GET LEGENDTITLE X , the variable X will be a string variable.

LEGENDTITLE is the title string that will appear centred above the legend frame, if LEGENDTITLEON is set on, and if LEGENDTITLEHEIGHT is > 0
