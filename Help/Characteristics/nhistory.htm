NHISTORY

Syntax: SET NHISTORY n
GET NHISTORY { n }
Default: NHISTORY = -1

NHISTORY controls how many lines of history to display for each numeric variable as a result of the SHOW command.

NHISTORYmeaning
< 0all stored history lines will be displayed
= 0,1only the latest history line will be displayed
> 0a maximum of NHISTORY lines of history will be displayed for each variable
