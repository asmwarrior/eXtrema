LEGENDTITLEFONT

Syntax: SET LEGENDTITLEFONT s
GET LEGENDTITLEFONT { s }
Default: LEGENDTITLEFONT = SANS

Note: The value of the LEGENDTITLEFONT characteristic is a string and not a numeric value. So, if you enter: GET LEGENDTITLEFONT X, the variable X will be a string variable.

LEGENDTITLEFONT controls the font of the graph legend title. The parameter f should be the font name.

Examples:

SET LEGENDTITLEFONT SANS
SET LEGENDTITLEFONT SERIF
