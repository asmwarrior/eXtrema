YLABELFONT

Syntax: SET YLABELFONT s
GET YLABELFONT { s }
Default: YLABELFONT = SANS

Note: The value of the YLABELFONT characteristic is a string instead of a numeric value. So, if you enter: GET YLABELFONT X , the variable X will be a string variable.

YLABELFONT controls the font of the automatic y-axis text label. The parameter f should be the font name.

Examples:

SET YLABELFONT SANS
SET YLABELFONT SERIF
