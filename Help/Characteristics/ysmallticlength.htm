YSMALLTICLENGTH

Syntax: SET YSMALLTICLENGTH n
SET %YSMALLTICLENGTH n
GET YSMALLTICLENGTH { n }
GET %YSMALLTICLENGTH { n }
Default: %YSMALLTICLENGTH = 1

YSMALLTICLENGTH controls the length of the optional small tic marks on the y-axis. These are the unlabeled tic marks between the large, numbered, tic marks. There are two forms:

 YSMALLTICLENGTH → the length, in world coordinate units, of the small tic marks on the y-axis
%YSMALLTICLENGTH → the length, expressed as a percentage of the height of the graphics sub-window, that is, YSMALLTICLENGTH = %YSMALLTICLENGTH
* (YUPPERWINDOW - YLOWERWINDOW)/100.
