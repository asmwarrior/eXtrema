TEXTINTERACTIVE

Syntax: SET TEXTINTERACTIVE n
GET TEXTINTERACTIVE { n }
Default value: TEXTINTERACTIVE = 1

If TEXTINTERACTIVE ≠ 0, then the graphics cursor is used to interactively place a text string when drawn by the TEXT command.

If TEXTINTERACTIVE = 0, then XTEXTLOCATION and YTEXTLOCATION will be used to determine the reference point for the character string.

In either case, the justification will be determined by the value of TEXTALIGN.
